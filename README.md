# 개요 (Overview)

- Servlet 기반으로 만들어진 맛집리뷰 사이트 입니다.

- css, js, font, editor는 문제가 있을것 같아 제외하였습니다.

# 주요 기능

- 로그인 : 회원만 이용가능
  
- 맛집 정보 : 맛집들의 정보 확인가능
  
- 맛집 리뷰 : 맛집들마다 리뷰 확인가능
  
- 맛집 지도 : 맛집들의 위치를 지도를 통해 확인가능
  
- 맛집 커뮤니티 : 커뮤니티를 통해 글쓰기 가능
  
- 고객문의 : 문의를 통해 문제 해결 가능
